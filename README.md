# open-google-cli

> Open a Google search from the terminal

## Installation

```
npm install --global open-google-cli
```

## Usage

```
$ open-google --help

  Usage
    $ open-google <search query>

  Examples
    $ free-space Kirkhorn
    $ free-space "Knut Kirkhorn"
    $ free-space "example search"
```
