# open-google-cli

> Open a Google search from the terminal

## Installation

```
npm install --global open-google-cli
```

## Usage

```
$ open-google --help

  Usage
    $ open-google <search query>

  Examples
    $ open-google Kirkhorn
    $ open-google Knut Kirkhorn
    $ open-google example search
```
